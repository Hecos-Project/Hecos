PC Automation
Enable PC Automation (Mouse, Keyboard, Windows)

Allows Hecos to control your mouse, keyboard, and application windows programmatically. Combine with WEBCAM → Desktop Screenshot for a full see-and-act loop.

FailSafe: Move your mouse to the top-left corner of the screen to instantly abort any automation action.

Mouse Settings
Smoothing duration for mouse movement (0 = instant). Default: 0.15s.
Keyboard Settings
Pause between each keystroke when typing text. Default: 0.02s.
Window Management
Enable Window Control (list, focus, minimize, maximize)

Requires pygetwindow. Allows the AI to identify and manage application windows by title.

OCR (Text Clicking)

The click_text tool requires Tesseract OCR installed on the system.

Windows: UB-Mannheim installer
macOS: brew install tesseract
Linux: sudo apt install tesseract-ocr
