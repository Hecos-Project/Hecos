Gestione Template

Crea messaggi predefiniti per WhatsApp, Telegram e Discord.
Puoi impostare un template come Default per formattare automaticamente le risposte di Hecos.

Telegram (Raccomandato)

Usa l'API ufficiale di Telegram per inviare messaggi in modo rapido e affidabile. Crea un bot tramite @BotFather.

Il token segreto rilasciato da @BotFather.
Usato come fallback se il comando non specifica il destinatario.
WhatsApp BETA

Hecos supporta l'invio sia tramite l'App Desktop ufficiale che tramite WhatsApp Web.

Opzione 1 (Consigliata): Installa l'App ufficiale di WhatsApp per Windows e fai l'accesso. Hecos rileverà l'app e invierà i messaggi molto più velocemente senza aprire nuove schede.
Opzione 2: Se l'App non è installata, Hecos aprirà in automatico una scheda nel tuo browser su WhatsApp Web. Assicurati di aver già scansionato il QR code per mantenere la sessione attiva.

Aggiunto in automatico ai numeri digitati senza prefisso (es. +39 per l'Italia).
Invia i testi multiriga e i template in un unico messaggio (anziché dividere in più messaggi continui).
Discord

Invia messaggi su un server Discord configurando un Incoming Webhook. Non richiede la creazione di un Web Bot.

Copia l'URL dalle Impostazioni Server → Integrazioni → Webhooks.
