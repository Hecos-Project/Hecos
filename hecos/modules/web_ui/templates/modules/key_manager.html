💡 Come funziona il failover:
1. Hecos tenta la chiave #1 con il timeout impostato
2. Se riceve 429/401/timeout → marca la chiave in cooldown e prende la chiave #2
3. Ripete fino a esaurimento tentativi o chiavi disponibili
4. Le chiavi in cooldown vengono automaticamente ripristinate dopo il tempo impostato
