Performance & Resources

On low-spec machines, Hecos SDK processes can consume significant CPU and RAM. Disabling the SDK will stop all hecos-sdk-* isolated subprocesses. HPM modules that require an isolated environment (e.g. Image Gen, Maps) will be unavailable until re-enabled. Shared-mode packages (Messenger, Templates, Mail) are not affected.
