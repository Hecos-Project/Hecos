🤖 Control Paradigm (PC & Browser)
Enable BROWSER Module
Completely toggle the AI's ability to use web browsers.

Choose how deeply Hecos should interact with your system. You can use an isolated internal browser (discreet), hook into your personal Chrome (Takeover), or give Hecos "eyes" to control the entire Windows OS with your real mouse.

1. Isolated AI Browser (App Mode)
Hecos uses its own dedicated Chromium browser. Great for quick background web searches without popping up windows that disturb you.
Does not use your saved passwords or browsing history.
2. Your Real Browser (Takeover CDP)
Hecos "undercover" uses your Google Chrome (or Edge). It sees your extensions, logins, and bookmarks.
⚠️ CRITICAL REQUIREMENT: You must launch Chrome using this shortcut parameter: chrome.exe --remote-debugging-port=9222. If Chrome is opened normally, the hook will fail for security reasons.
3. Physical Mouse + Vision (PC Automation)
Hecos uses AI to see the screen (via periodic screenshots) and moves your real mouse pointer on the current page or in any desktop program.
Note: This is the ultimate human action. If enabled, you cannot use the PC while Hecos is "driving" because it will move the physical mouse.
