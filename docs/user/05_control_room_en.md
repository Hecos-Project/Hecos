# 🎛️ Control Room

> *"A dedicated masonry-style dashboard where multiple interactive widgets run simultaneously."*

![Hecos - Control Room](https://github.com/Hecos-Project/Hecos-Assets/blob/main///006_Hecos_ControlRoom_Standalone.png?raw=true)


> — Hecos WebUI Tour

The **Control Room** is one of the five main pillars of the Hecos WebUI. It is a dedicated, fullscreen dashboard where you can run and arrange multiple interactive **widgets** side by side in a responsive grid layout.

## What is the Control Room?

Unlike the main Chat, which is conversation-focused, the Control Room gives you a permanent, at-a-glance view of your system and data. Think of it as your **mission control panel**: a space you can fill with exactly the information you care about.

## How to Open It

You can access the Control Room from:
- The **sidebar** of the main Chat (click the grid icon).
- Or via the keyboard shortcut shown in the topbar.

## Widget Grid

The Control Room uses a **drag-and-drop masonry grid** (powered by GridStack). You can:
- **Add widgets** from the Package Library.
- **Resize** any widget by dragging its corner.
- **Reorder** widgets freely — the layout is saved automatically.
- **Remove** a widget from the room without uninstalling it.

## Included Widgets (Examples)

| Widget | Description |
|--------|-------------|
| **OS Telemetry** | Real-time CPU, RAM, and VRAM usage graphs. |
| **Weather Pro** | Live weather data and 7-day forecast. |
| **Calendar** | Upcoming events and reminders at a glance. |
| **Quick Links** | Your favorite links, one click away. |
| **Voice Visualizer** | Animated visualization when Hecos speaks. |
| **Webcam Feed** | Live camera feed from your local webcam. |

## Adding New Widgets

New widgets can be installed directly from the **Package Library** (Central Hub → Packages → Store). Once installed, they automatically appear in the "Add Widget" panel inside the Control Room.
