# ⚙️ Central Hub

> *"Il centro di configurazione e gestione. Installa pacchetti, regola le impostazioni e gestisci gli utenti."*

![Hecos - Central Hub](https://github.com/Hecos-Project/Hecos-Assets/blob/main////009_HecosCentralHub_003.png?raw=true)

Il **Central Hub** è il centro di comando di Hecos. Accessibile tramite l'icona dell'ingranaggio nella WebUI o premendo **F7**, ti permette di riconfigurare il sistema in tempo reale senza dover modificare complessi file YAML.

## Funzionalità
- **Configurazione Dinamica:** Cambia il Modello IA, le impostazioni Vocali e la Personalità (Anima) all'istante.
- **Gestore Pacchetti (HPM):** Un tab dedicato "Pacchetti" ti permette di esplorare lo Store online, scaricare nuovi widget, plugin e app, e installarli con un singolo clic.
- **Gestione Utenti:** Gestisci profili locali, vault e permessi.
- **Design Simmetrico:** Passa fluidamente tra una classica vista a Schede e una vista globale "Wall" (masonry) per un controllo totale.
