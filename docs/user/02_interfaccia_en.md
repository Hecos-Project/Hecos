# 🖥 2. User Interface (Safe Scrolling UI)

Zentra's terminal interface uses an anchored architecture:
- **Dashboard (Plugins):** Live hardware stats (CPU, RAM, VRAM).
- **Status Bar (System):** Shows current state (PRONTO, PENSANDO, PARLANDO), active model, and personality.
- **Chat Area:** Interaction history scrolls from row 7 downwards, keeping the dashboard visible.
