# 👥 18. Gestión Multi-Usuario y Perfiles
Hecos soporta múltiples usuarios independientes, cada uno con sus propios datos y configuraciones aislados.

## Características
- **Memorias Aisladas**: Las conversaciones y el contexto de aprendizaje son privados para cada usuario.
- **Bóvedas Personales**: Almacenamiento dedicado para archivos y recursos ubicado en `memory/vaults/username`.
- **Avatares Personalizados**: Personaliza tu identidad dentro de la interfaz de chat.
- **Acceso Basado en Roles**: Distingue entre usuarios Admin e Invitado (Guest).

## Administración
Los administradores pueden gestionar los usuarios desde la pestaña **Users** en el Panel de Configuración.
- **Crear Usuario**: Añade nuevas cuentas al sistema.
- **Actualizar Perfiles**: Cambia nombres de usuario, contraseñas y roles.
- **Eliminar Usuario**: Elimina completamente a un usuario y su bóveda asociada.
