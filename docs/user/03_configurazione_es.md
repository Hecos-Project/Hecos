# ⚙️ 3. Configuración Dinámica (O-T-F)

Utiliza las teclas de función (F1-F7) para interactuar con `config.json` en tiempo real.

* **[ F1 ] Manual de Acción (Ayuda)**
* **[ F2 ] Cambiar Modelo de IA**
* **[ F3 ] Cambiar Personalidad (Alma)**
* **[ F4 ] Activar/Desactivar Micrófono**
* **[ F5 ] Activar/Desactivar Voz (TTS)**

### 🎛️ El Panel de Control [ F7 ]
Ofrece control granular sobre el motor Zentra Core. Los cambios se pueden descartar (ESC) o guardar con un **Reinicio en Frío** automático.
