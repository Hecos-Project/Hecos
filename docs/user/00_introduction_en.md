# 🪐 Welcome to Hecos

> 💡 Hecos is your everyday companion. You can talk to it about whatever you want; it can control your computer, send messages, and with its native personality-switching feature, it can act as a butler, a friend, or even your virtual partner. Need it to work without AI? No problem — Hecos runs powerful automation flows, responds to direct commands, and manages your calendar, reminders, messenger platforms, mail accounts, and lists completely on its own.

🛠️ **Under the Hood (For Developers & Power Users):** Now advancing through `v0.36.0 Alpha`, Hecos has evolved into a fully fledged, distributed modular platform. The ecosystem features a decoupled **Client-Server architecture** consisting of a headless CLI Core (the backend server) and an advanced WebUI (the client). The system synchronizes an interactive Chat view, a real-time **Control Room** powered by live telemetry widgets, and **Hecos Flows**—a visual automation engine that executes workflows with or without LLM overhead. 

Infrastructure utilities include a unified configuration panel (**Central Hub**), a secure remote file manager, an integrated proxy routing system, granular **Direct Commands**, dynamic Voice/VAD pipelines, and a centralized backup engine. Extensions are handled natively through the **Hecos Package Manager (HPM)**, fetching standalone `.hpkg` components directly from the newly integrated **Hecos Store**.

## 👤 What is Hecos?
More than a simple chatbot, Hecos is an **Agentic Layer** for your computer. Following the "LLM-as-CPU" paradigm, Hecos acts as the master orchestrator, managing memory, specialized tools, and cognitive profiles to assist you in complex workflows—from advanced file management to local code execution.

## 🧠 Core Architecture: Modular & Extensible
The heart of Hecos lies in its **Modularity**. Every function—be it a specialized Vision engine, a Web searcher, or a custom Tool—is a pluggable component. This allows Hecos to grow and adapt to the rapidly evolving AI landscape without legacy constraints. 

Extensions, apps, and patches are bundled into standalone `.hpkg` packages distributed via the ecosystem.

## 🗣️ Multi-Modal Interaction & Context
Hecos bridges the gap between human and machine through high-fidelity interfaces and native context handling:
- **Neural Voice Pipeline**: Using low-latency "Push-to-Talk" and advanced TTS/VAD, you can operate the system entirely via voice.
- **Cognitive Personas**: Hecos can swap its "Persona" or cognitive profile instantly, adapting its tone, knowledge base, and tool-access permissions.
- **RAG Semantic Memory**: An advanced vector-based semantic memory vault that allows the system to store, recall, and contextualize interactions securely and permanently.

## 🔌 Built-in System Modules
Hecos includes native, robust modules ready out of the box to control your workspace:
- **PC Automation & Browser Automation**: Deep operating system bridging and browser control execution loops to handle tasks, extract data, and script actions natively.
- **Messenger Bridges & Mail Integrations**: Dedicated modules to orchestrate incoming and outgoing communications across multiple chat networks and email setups seamlessly.
- **Centralized Backup** — A one-click solution to safely store your entire configuration, workspace files, installed modules, and memories.
- **Hecos Proxy** — An integrated local routing proxy to bypass web restrictions and securely connect isolated modules to the internet.

## 🔒 Privacy and Local Intelligence
Unlike many popular assistants, Hecos puts your privacy first. Many of its features are designed to run directly on your computer (**locally** via tools like Ollama or KoboldCpp). This means your data, semantic memory vaults, and conversations stay with you, without necessarily traveling over the internet.
