# 👁️ 5. Vision & Multimodal Interaction

Version 0.9.9 introduces native **Vision Support**.
- **Image Upload**: Drag & drop or paste (Ctrl+V) into the WebUI.
- **Multimodal AI**: Describe, analyze, and read text from images using Gemini, GPT-4o, or LLaVA.
- **Visual Feedback**: Real-time thumbnails in the chat bubble and attachment bar.
