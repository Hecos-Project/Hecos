# 🖥 2. Interfaz de Usuario (TUI)

La interfaz de terminal de Zentra utiliza una arquitectura anclada:
- **Dashboard (Plugins):** Estadísticas de hardware en tiempo real.
- **Barra de Estado:** Muestra el estado actual (PRONTO, PENSANDO, PARLANDO), el modelo activo y la personalidad.
- **Área de Chat:** El historial de interacción se desplaza desde la fila 7 hacia abajo.
