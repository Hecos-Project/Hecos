# ⚙️ Central Hub

> *"El centro de configuración y gestión. Instala paquetes, ajusta la configuración y gestiona usuarios."*

El **Central Hub** es el centro de mando de Hecos. Accesible a través del icono de engranaje en la WebUI o presionando **F7**, te permite reconfigurar el sistema sobre la marcha sin editar archivos YAML complejos.

## Características
- **Configuración Dinámica:** Cambia el Modelo de IA, los ajustes de Voz y la Personalidad (Alma) al instante.
- **Gestor de Paquetes (HPM):** Una pestaña dedicada a "Paquetes" te permite explorar la Tienda en línea, descargar nuevos widgets, complementos y aplicaciones, e instalarlos con un solo clic.
- **Gestión de Usuarios:** Gestiona perfiles locales, bóvedas (vaults) y permisos.
- **Diseño Simétrico:** Alterna sin problemas entre una vista clásica de Pestañas y una vista global de "Muro" (masonry) para un control total.
