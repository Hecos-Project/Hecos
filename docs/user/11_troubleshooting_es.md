# 🛠️ 11. Resolución de Problemas

1. **Interferencia Gráfica (Dashboard):** Se resuelve automáticamente al unirse al hilo de la interfaz de usuario durante las llamadas al menú F7.
2. **Logs:** Ubicados en `/logs`. Se pueden ocultar del chat a través de F7 → SYSTEM para una mejor lectura.
3. **Bucle de Activación de Audio:** Ajusta el `Energy Threshold` en **F7 → Listening** para calibrar el ruido ambiental.
