# 🎨 7. Generación de Imágenes

Zentra puede crear contenido visual utilizando el plugin `IMAGE_GEN`.
- **Cómo usarlo**: Simplemente pide a Zentra "Genera una imagen de..." o "Dibuja un...".
- **Servidores Externos**: Por defecto utiliza **Pollinations.ai** para una generación rápida y sin filtros.
- **Interacción**: Las imágenes generadas aparecerán directamente en el chat con opciones de zoom y descarga.
