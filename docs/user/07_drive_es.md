# 📁 Hecos Drive

> *"Un explorador de archivos y editor de código integrados. Tu IA tiene acceso directo a tu espacio de trabajo."*
> — Tour WebUI Hecos

**Hecos Drive** es uno de los cinco pilares principales de la WebUI de Hecos. Es un administrador de archivos y editor de código integrado que te da tanto a ti como a la IA acceso directo y contextual a los archivos en tu espacio de trabajo (workspace).

## ¿Qué es Hecos Drive?

Hecos Drive no es solo un navegador de archivos: es el puente entre la IA y tu sistema de archivos. Cuando Hecos necesita leer, escribir o editar un archivo durante una tarea, utiliza el Drive como su espacio de trabajo. Puedes seguir todo el proceso en tiempo real.

## Características Principales

### 📂 Explorador de Archivos
- Navega por carpetas y archivos dentro de tu espacio de trabajo configurado.
- Crea, renombra, mueve y elimina archivos directamente desde la interfaz.
- Arrastra archivos al Chat para enviarlos a Hecos para su análisis.

### ✏️ Editor de Código Integrado
- **Resaltado de sintaxis** para más de 50 lenguajes de programación.
- **Navegación rápida** entre archivos abiertos.
- **Vista previa en vivo** para archivos Markdown.
- Permite la edición directa de los archivos de configuración de Hecos (por ejemplo, flujos, personalidades).

### 🤖 Espacio de Trabajo Consciente de IA (AI-Aware)
- Cuando le pides a Hecos que *"cree un script en Python"* o *"edite el archivo de configuración"*, escribe directamente en el Drive.
- Puedes abrir inmediatamente cualquier archivo generado por la IA en el editor para revisarlo o ajustarlo.

## Cómo Acceder

Se puede acceder al Drive desde la navegación del **Central Hub** (menú superior → pestaña Drive) o mediante el comando directo:
```
/drive
```

---
*Hecos Drive te mantiene al mando: siempre ves exactamente lo que la IA lee y escribe.*
