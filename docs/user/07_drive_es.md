# 📁 8. Hecos Drive

> *"Un explorador de archivos y editor de código integrado. Su IA tiene acceso directo a su espacio de trabajo."*

![Hecos - Drive](https://github.com/Hecos-Project/Hecos-Assets/blob/main////////015_Hecos_Drive_FileManager.png?raw=true)

> — Tour WebUI Hecos

**Hecos Drive** es uno de los cinco pilares principales de la WebUI de Hecos. Es un administrador de archivos y editor de código integrado que le brinda tanto a usted como a la IA acceso directo y contextual a los archivos en su espacio de trabajo.

## ¿Qué es Hecos Drive?

Hecos Drive no es solo un navegador de archivos: es el puente entre la IA y su sistema de archivos. Cuando Hecos necesita leer, escribir o editar un archivo durante una tarea, utiliza Drive como su espacio de trabajo. Puede seguir el proceso en tiempo real.

![Hecos - Drive ](https://github.com/Hecos-Project/Hecos-Assets/blob/main/////////016_Hecos_Drive_FileManager_002.png?raw=true)


## Características Clave

### 📂 Explorador de Archivos
- Navegue por carpetas y archivos dentro de su espacio de trabajo configurado.
- Cree, renombre, mueva y elimine archivos directamente desde la interfaz.
- Arrastre archivos al Chat para enviarlos a Hecos para su análisis.

### ✏️ Editor de Código Integrado
- **Resaltado de sintaxis** para más de 50 lenguajes de programación.
- **Navegación rápida** entre archivos abiertos.
- **Vista previa en vivo** para archivos Markdown.
- Admite la edición directa de archivos de configuración de Hecos (por ejemplo, flujos, personas).

### 🤖 Espacio de Trabajo con IA
- Cuando le pide a Hecos que *"cree un script de Python"* o *"edite el archivo de configuración"*, escribe directamente en Drive.
- Puede abrir cualquier archivo generado por IA inmediatamente en el editor para revisarlo o ajustarlo.

## Cómo Acceder

Se puede acceder a Drive desde la navegación del **Central Hub** (menú superior → pestaña Drive) o mediante comando directo:
```
/drive
```
