# ⚡ Inizio Rapido (Quick Start)

Benvenuto in Hecos! Segui questi passaggi per configurare il sistema e iniziare subito a usare l'IA sul tuo PC.

## 1. Installazione (Bootstrap)
Il modo più semplice per iniziare è usare gli script di configurazione automatica nella cartella principale (root):
- **Windows:** Fai doppio click su `START_SETUP_HERE_WIN.bat`
- **Linux:** Apri il terminale ed esegui `bash START_SETUP_HERE_LINUX.sh`

Questi script installeranno automaticamente le dipendenze e avvieranno il **Setup Wizard** nel tuo browser.

## 2. Il Wizard di Setup
Al primo avvio, il browser si aprirà all'indirizzo `http://localhost:7070`. Segui i passaggi guidati:
1. **Benvenuto**: Clicca su "Get Started".
2. **Lingua**: Seleziona "Italiano" (o la lingua che preferisci).
3. **Scegli il Cervello (AI Provider)**: 
   - **Cloud (Online)**: Usa modelli potenti come Gemini o GPT-4o. In questo caso dovrai inserire la tua **Chiave API**.
   - **Locale (Offline)**: Se hai Ollama o KoboldCpp installato sul tuo PC, Hecos si collegherà automaticamente. **In questo caso NON serve alcuna chiave API**, tutto gira sul tuo hardware!
4. **Generazione Immagini**: Al momento la creazione di immagini richiede un provider online. Il modo migliore e più veloce è creare un account gratuito su **HuggingFace**, generare un "Access Token" e inserirlo nelle impostazioni di Hecos per usare modelli avanzati come **FLUX.1-dev**.
5. **Configura Personalità**: Scegli "l'anima" del tuo assistente (es. Urania o Atlas).
6. **Fine**: Clicca su "Save and Start".

## 3. Primo Utilizzo
Ora che Hecos è attivo, ecco come interagire:
- **Chat**: Scrivi nella barra di testo in basso nella WebUI e premi Invio.
- **Voce**: 
  - Clicca l'icona del microfono nella WebUI.
  - Oppure usa la scorciatoia globale **Ctrl+Shift+Z** (Windows) per parlare senza nemmeno aprire il browser.
- **Visione**: Trascina un'immagine nella chat per chiedere a Hecos di descriverla o analizzarla.

## 4. Pannello di Controllo (F7)
Per cambiare parametri, aggiungere nuove chiavi API o attivare plugin:
- Premi **F7** sulla tastiera o clicca sull'icona dell'ingranaggio/logo nella WebUI per aprire lo **Hecos Hub**.
- Le modifiche vengono salvate istantaneamente.

## 5. Icona Tray (Background)
Hecos resta attivo nella barra delle icone (vicino all'orologio di Windows). 
- Puoi chiudere la scheda del browser: il sistema continuerà a girare in sottofondo per rispondere ai tasti di scelta rapida.
- Clicca col tasto destro sulla "Z" per riaprire la WebUI o chiudere definitivamente Hecos.
