# 🔄 6. Gestione delle Risposte

Hecos offre strumenti avanzati per gestire il flusso della conversazione.

- **Rigenerazione**: Se non sei soddisfatto di una risposta, usa il pulsante con la freccia circolare per chiedere a Hecos di riprovare. La vecchia risposta verrà sostituita da una nuova elaborazione.
- **Interruzione**: Puoi fermare la generazione di una risposta o la sintesi vocale in qualsiasi momento premendo il tasto `ESC` o il pulsante di stop nella barra laterale.
- **Contesto Dinamico**: Hecos mantiene una memoria della conversazione corrente per garantire risposte coerenti e pertinenti.
