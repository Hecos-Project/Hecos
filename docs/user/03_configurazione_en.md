# ⚙️ 3. Dynamic Configuration (O-T-F)

Use Function Keys (F1-F7) for real-time interaction with `config.json`.

* **[ F1 ] Action Manual (Help)**
* **[ F2 ] Change AI Model**
* **[ F3 ] Change Personality (Soul)**
* **[ F4 ] Toggle Microphone**
* **[ F5 ] Toggle Voice (TTS)**

### 🎛️ The [ F7 ] Control Panel
Provides granular control over the Zentra Core engine via a Curses-based menu. Changes can be discarded (ESC) or saved with an automatic **Cold Reboot**.
