# 🔄 6. Gestión de Respuestas

Hecos ofrece herramientas avanzadas para gestionar el flujo de la conversación.

- **Regeneración**: Si no estás satisfecho con una respuesta, usa el botón de la flecha circular para pedir a Hecos que intente de nuevo. La respuesta antigua será reemplazada por una nueva generación.
- **Interrupción**: Puedes detener la generación de una respuesta o la síntesis de voz en cualquier momento presionando `ESC` o il botón de parada en la barra lateral.
- **Contexto Dinámico**: Hecos mantiene una memoria de la conversación actual para garantizar respuestas coherentes y pertinentes.
