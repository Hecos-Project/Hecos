# ⚙️ Central Hub

> *"The configuration and management center. Install packages, tune settings, and manage users."*

![Hecos - Central Hub](https://github.com/Hecos-Project/Hecos-Assets/blob/main///008_HecosCentralHub_002.png?raw=true)

The **Central Hub** is the command center of Hecos. Accessible via the gear icon in the WebUI or by pressing **F7**, it allows you to reconfigure the system on the fly without editing complex YAML files.

## Features
- **Dynamic Configuration:** Change the AI Model, Voice settings, and Personality (Soul) instantly.
- **Package Manager (HPM):** A dedicated "Packages" tab allows you to browse the online Store, download new widgets, plugins, and apps, and install them with a single click.
- **User Management:** Manage local profiles, vaults, and permissions.
- **Symmetrical Design:** Toggle seamlessly between a classic Tab view and a global "Wall" masonry view for total oversight.
