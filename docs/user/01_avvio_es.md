# 🚀 1. Arranque y Controles Iniciales

Al iniciar el ejecutable o el script de Python, Zentra comienza su secuencia de **Arranque Sincronizado**.

### Diagnóstico Previo
Por defecto, el sistema verifica:
- Integridad de carpetas vitales.
- Estado del hardware (CPU y RAM).
- Estado de los módulos de audio y voz.
- Respuesta del servidor de IA.
- Escaneo de plugins activos/desactivados.

### ⚡ Arranque Rápido (Fast Boot)
Puedes activar el **Arranque Rápido** en el panel **F7** bajo `SYSTEM` para saltar el diagnóstico y reducir el tiempo de carga a **~0.5 segundos**.
