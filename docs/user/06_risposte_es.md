# 🔄 6. Gestión de Respuestas

- **Regenerar Respuesta**: Usa el botón con la flecha circular al lado de cada mensaje de la IA para pedir a Zentra que lo intente de nuevo. La respuesta anterior se eliminará y se activará una nueva inferencia.
- **Mensajería Interna**: La regeneración utiliza un canal de API directo, preservando el prompt original y su contexto.
