# 🤖 12. Agente Autónomo y Sandbox

Hecos está equipado con un motor de razonamiento avanzado que le permite operar como un Agente autónomo.

- **Ciclo de Agente**: Hecos puede planificar y ejecutar tareas complejas dividiéndolas en subobjetivos, llamando a plugins y herramientas en sucesión.
- **Sandbox AST**: Cada vez que la IA genera código (Python, algoritmos, lógica), este se ejecuta en un "vivero" protegido (Sandbox AST). Esto evita la ejecución de comandos peligrosos para el sistema operativo.
- **Herramientas Automáticas**: El Agente puede usar la calculadora, gestionar archivos y buscar información sin la intervención constante del usuario.
