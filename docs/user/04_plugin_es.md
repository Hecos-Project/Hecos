# 🔌 4. Sistema Modular / Plugins

Zentra es infinitamente ampliable a través del directorio `plugins/`.
- **Arquitectura de Extensiones JIT**: Extensiones cargadas bajo demanda (ej. Editor de Código Zentra).
- **Drive Pro**: Navegación completa del sistema de archivos con seguridad de ruta absoluta.
- **WebUI Nativa**: Interfaz basada en navegador para chat y configuración.
