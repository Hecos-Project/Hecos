# 👁️ 5. Visión e Interacción Multimodal

La versión 0.9.9 introduce **Soporte de Visión** nativo.
- **Carga de Imágenes**: Arrastra y suelta o pega (Ctrl+V) en la WebUI.
- **IA Multimodal**: Describe, analiza y lee texto de imágenes usando Gemini, GPT-4o o LLaVA.
- **Feedback Visual**: Miniaturas en tiempo real en la burbuja de chat y barra de adjuntos.
