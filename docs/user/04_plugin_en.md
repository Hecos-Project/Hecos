# 🔌 4. Modular System / Plugins

Zentra is infinitely expandable via the `plugins/` directory.
- **JIT Extension Architecture**: Lazy-loaded extensions (e.g., Zentra Code Editor).
- **Drive Pro**: Full filesystem traversal with absolute path security.
- **Native WebUI**: Browser-based interface for chat and configuration.
- **Clean Disabling**: Faulty modules can be bypassed without crashing the system.
