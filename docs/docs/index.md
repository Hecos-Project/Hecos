# Hecos Documentation

Welcome to the official documentation for **Hecos**, your personal, offline AI assistant.

Hecos is designed to be a private, modular, and powerful companion that runs entirely on your hardware.

## What is Hecos?

Hecos combines local LLMs (Large Language Models), voice interaction, system automation, and a modular architecture to provide a fully customizable AI experience without cloud dependencies.
