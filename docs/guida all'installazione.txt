===================================================
 GUIDA ALL'INSTALLAZIONE DI HECOS CORE v0.19.1
          (Per PC "vergine" - da zero)
===================================================

1. INSTALLARE PYTHON (OBBLIGATORIO)
   - Vai su https://www.python.org/downloads/
   - Scarica Python 3.10 o superiore (consigliato 3.11/3.12)
   - Durante l'installazione, spunta "Add Python to PATH"

2. AVVIO RAPIDO (CONSIGLIATO)
   - Estrai Hecos in una cartella
   - Esegui lo script di bootstrap per la tua piattaforma:
     * Windows: Doppio click su START_SETUP_HERE_WIN.bat
     * Linux: Esegui bash START_SETUP_HERE_LINUX.sh
   - Questo script installerà le dipendenze e avvierà il Wizard di Setup nel browser.

3. AVVIO MANUALE (PER UTENTI AVANZATI)
   - Installa dipendenze: pip install -r requirements.txt
   - Avvia sistema completo: python main.py
   - Avvia solo WebUI: HECOS_WEB_RUN_WIN.bat (Win) o hecos_web_run.sh (Linux)
   - Avvia solo Console: HECOS_CONSOLE_RUN_WIN.bat (Win) o HECOS_CONSOLE_RUN.sh (Linux)

4. ICONA TRAY E BACKGROUND
   - Per far girare Hecos in background e gestirlo comodamente:
     * Avvia Hecos normalmente; l'icona apparirà nella barra di sistema (Tray).
     * Dall'icona puoi aprire la WebUI, la Console o chiudere il sistema.
     * Usa START_HECOS_TRAY_WIN.bat se l'icona sparisce accidentalmente.

5. LOGIN DEFAULT
   - User: admin / Pass: hecos  (Cambiala subito nel Pannello di Controllo!)

6. MCP & TOOL HUB
   - Vai in Configurazione -> MCP Bridge per installare nuovi tool da Smithery o GitHub.
