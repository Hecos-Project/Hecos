# 🏗️ 1. Arquitectura de Sistema

Zentra Core está construido sobre una **Arquitectura Modular Orientada a Objetos**.
- **Patrón Singleton**: Los gestores principales coinciden en un estado consistente.
- **Ejecución Asíncrona**: Hilos en segundo plano para tareas pesadas.
- **Carga Diferida (Lazy Loading)**: Los plugins cargan extensiones solo cuando son necesarias.
