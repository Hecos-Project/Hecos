🔗 Extension di Zentra Drive — richiede il modulo Drive attivo. Apri Drive →
🖼️ Drive Media Viewer — Configurazione

Visualizzatore multimediale integrato in Zentra Drive. Supporta immagini (JPG, PNG, WebP, GIF, SVG), video (MP4, WebM, MKV), e audio (MP3, OGG, WAV, FLAC).

🖼️ Immagini
Mostra metadati immagine (EXIF)

Mostra informazioni EXIF: fotocamera, data di scatto, GPS, ecc.

Modalità Slideshow automatica

Avvia automaticamente la presentazione quando si entra in una cartella di immagini.

🎬 Video
Autoplay video

Avvia automaticamente il video all'apertura (senza audio per policy browser).

Loop video

Ripete il video in loop al termine.

Mostra controlli player

Mostra play/pausa, barra di avanzamento, volume e fullscreen.

🎵 Audio
Autoplay audio

Avvia automaticamente i file audio all'apertura.

Mostra visualizzatore waveform

Mostra la forma d'onda animata durante la riproduzione audio.

🔧 Generale
File con queste estensioni aprono il viewer immagini. Separati da virgola.
File con queste estensioni aprono il player video.
File con queste estensioni aprono il player audio.
ℹ️ Formati e compatibilità
La riproduzione dipende dalle capacità del browser. Per MKV e AVI considera la conversione a MP4/WebM.
I file media vengono serviti via streaming da Drive (GET /api/drive/stream/<path>).
