📡 Remote Triggers — PTT Input Bus

Activate Push-to-Talk from any source: keyboard shortcut, smartwatch / Bluetooth media keys, HTTP webhooks, or a custom hotkey. Toggle each source independently. The audio module reads a single shared state — no conflicts.

⌨️ Keyboard Hotkey

Hold to talk. Hotkey configured in Voice settings (ptt_hotkey). Enabled by default.

⌚ Smartwatch / Bluetooth Media Keys

Uses the Play/Pause media key. Compatible with smartwatches (H13, etc.) and Bluetooth headsets. Toggle mode: first press starts recording, second press stops.

🎯 Custom Key (Hold-to-Talk)

Any single key or combo (e.g. f8, ctrl+alt+space). Hold to record.

🌐 HTTP Webhook (Arduino / ESP32 / Automation)

External devices can trigger PTT via GET/POST request. No authentication required for local network use.

Start: GET /api/remote-triggers/ptt/start
Stop: GET /api/remote-triggers/ptt/stop
Toggle: GET /api/remote-triggers/ptt/toggle
