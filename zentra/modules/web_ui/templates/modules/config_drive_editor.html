📝 Drive Editor — Configurazione

Editor di codice e testo integrato in Zentra Drive. Supporta syntax highlighting per oltre 50 linguaggi, navigazione rapida e anteprima live.

⚙️ Comportamento
Auto-Save

Salva automaticamente il file durante la modifica (ogni 30 secondi).

Conferma prima di chiudere con modifiche non salvate

Avvisa se si tenta di navigare via con modifiche pendenti.

Backup automatico prima del salvataggio

Crea un file .bak nella stessa cartella prima di sovrascrivere.

🎨 Vista Editor
Mostra numeri di riga

Word Wrap (a capo automatico)

Se attivo, le righe lunghe vengono spezzate visualmente senza modificare il file.

Minimap laterale

Preview compatta del file a destra dell'editor.

🔒 Limiti
File più grandi vengono aperti in modalità sola lettura.
Lista separata da virgola. Questi file si apriranno solo in anteprima/sola lettura.
ℹ️ Note tecniche
L'editor usa Monaco Editor (la base di VS Code).
Il syntax highlighting è automatico in base all'estensione del file.
I file vengono salvati via API Drive (POST /api/drive/save).
