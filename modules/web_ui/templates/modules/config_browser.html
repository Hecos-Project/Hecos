Modalità Browser

Scegli come Hecos interagisce con il browser. La modalità CDP è consigliata perché usa il tuo Chrome reale con le tue sessioni attive.

🎯 Il Tuo Browser Reale (CDP Takeover) CONSIGLIATO
Hecos si aggancia al tuo Google Chrome in modo invisibile. Vede le tue estensioni, sessioni di login e preferiti.
Chrome deve essere avviato con: chrome.exe --remote-debugging-port=9222
🧪 Browser Isolato (App Mode)
Hecos usa un browser Chromium dedicato, separato dal tuo. Nessun accesso a password o sessioni salvate.
Richiede Playwright — il browser viene scaricato automaticamente al primo avvio.
